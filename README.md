# Machine-Learning-Based-Analysis-of-Educational-Data-using-Multiple-Algorithms
📖 Project Description
<br>
This project focuses on analyzing a school dataset using multiple Machine Learning algorithms to extract meaningful insights, perform predictions, and identify hidden patterns.

The main objective of this project is to apply different types of machine learning techniques—regression, classification, and association rule mining—on educational data to understand factors affecting student performance and school characteristics.

The dataset includes features such as teacher-student ratio, parent literacy rate, internet availability, and infrastructure facilities. These features are used to build predictive models and analyze relationships within the data.

⚙️ Machine Learning Algorithms Used
<br>
This project uses a combination of algorithms for different types of tasks:
<br>
🔹 Regression Algorithms:
<br>
*Simple Linear Regression → Predicts Math Score
<br>
*Multiple Linear Regression → Predicts Science Score
<br>
<br>
🔹 Classification Algorithms:
<br>
*Decision Tree → Classifies student performance
<br>
*Logistic Regression → Binary classification of performance
<br>
*Support Vector Machine (SVM) → Classifies schools (Urban/Rural)
<br>
*K-Nearest Neighbors (KNN) → Classification based on nearest data points
<br>
*Naive Bayes → Probability-based classification
<br>
<br>
🔹 Association Rule Algorithm:
<br>
Apriori → Finds relationships between school facilities
